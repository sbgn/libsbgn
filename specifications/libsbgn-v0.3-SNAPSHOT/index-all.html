arc - element of namespace http://sbgn.org/libsbgn/0.3

The arc element describes an SBGN arc between two SBGN nodes.

 
arcgroup - element of namespace http://sbgn.org/libsbgn/0.3

The arc group describes a set of arcs and glyphs that together have a relation.

 
bbox - element of namespace http://sbgn.org/libsbgn/0.3

The bbox element describes a rectangle.

 
callout - nested element in element of namespace http://sbgn.org/libsbgn/0.3

The callout element is only used for glyphs with class annotation.

 
clone - nested element in element of namespace http://sbgn.org/libsbgn/0.3

The clone element (which is optional) means the glyph carries a clone marker.

 
end - nested element in element of namespace http://sbgn.org/libsbgn/0.3

The end element represents the ending point of the arc's path.

 
entity - nested element in element of namespace http://sbgn.org/libsbgn/0.3

The entity is only used in activity flow diagrams.

 
extension - nested element in complexType of namespace http://sbgn.org/libsbgn/0.3
 
glyph - element of namespace http://sbgn.org/libsbgn/0.3

The glyph element is:

  • either a stand-alone, high-level SBGN glyph (EPN, PN, compartment, etc),
  • or a sub-glyph (state variable, unit of information, inside of a complex, ...)

 
label - element of namespace http://sbgn.org/libsbgn/0.3

The label element describes the text accompanying a glyph.

 
map - element of namespace http://sbgn.org/libsbgn/0.3

The map element describes a single SBGN PD map.

 
next - nested element in element of namespace http://sbgn.org/libsbgn/0.3

The next element represents the next point in the arc's path.

 
notes - nested element in complexType of namespace http://sbgn.org/libsbgn/0.3
 
point - element of namespace http://sbgn.org/libsbgn/0.3

The point element is characterized by PointAttributes, which describe absolute 2D cartesian coordinates.

 
PointAttributes - attributeGroup of namespace http://sbgn.org/libsbgn/0.3

The PointAttributes group describes absolute 2D cartesian coordinates.

 
port - element of namespace http://sbgn.org/libsbgn/0.3

The port element describes an anchor point to which arcs can refer as a source or target.

 
sbgn - element of namespace http://sbgn.org/libsbgn/0.3

The sbgn element is the root of any SBGNML document.

 
SBGNBase - complexType of namespace http://sbgn.org/libsbgn/0.3

The SBGNBase type is the base type of all main components in SBGN.

 
start - nested element in element of namespace http://sbgn.org/libsbgn/0.3

The start element represents the starting point of the arc's path.

 
state - nested element in element of namespace http://sbgn.org/libsbgn/0.3

The state element should only be used for state variables.
